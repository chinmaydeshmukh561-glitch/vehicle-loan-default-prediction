# vehicle-loan-default-prediction
"Credit risk model predicting vehicle loan defaults using real Indian NBFC data (LTFS) — Logistic Regression vs. Random Forest, with feature-level interpretation of default drivers."

use this link to download the train.csv file
Link: https://www.kaggle.com/datasets/mamtadhaker/lt-vehicle-loan-default-prediction
create a kaggle account and Click the Download button (top right) — this gets you a zip file with train.csv (and possibly test.csv, data_dictionary.csv)

Note:- While running the code chnage the path in the code to your respective path where you extracted the train.csv file otherwise the code will not run and will show the error
